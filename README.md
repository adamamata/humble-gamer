# Humble Gamer
## Description
Humble Gamer is a CRUD web application that uses the RAWG API to search and store your favourite games. 

## MVP
- CRUD for published games. 
- Users can comment on each game. 
- Authentication. 
- Search bar / filter through games (using API).
